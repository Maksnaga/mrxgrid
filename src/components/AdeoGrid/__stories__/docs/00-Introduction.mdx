# MrxGrid

<div className="badges">
  <span className="badge badge--green">Vue 3.5</span>
  <span className="badge badge--blue">TypeScript strict</span>
  <span className="badge badge--blue">Vite 8</span>
  <span className="badge badge--amber">Mozaic Design System</span>
</div>

MrxGrid est une bibliothèque de data grid haute performance pour Vue 3, conçue
pour gérer plusieurs centaines de milliers de lignes sans dégrader l'expérience
utilisateur. Elle vise la parité fonctionnelle avec les leaders du marché
(AG&nbsp;Grid, Handsontable) tout en s'intégrant nativement avec
`@mozaic-ds/vue`.

## Pourquoi MrxGrid ?

  <div className="feature-card">

  <div className="feature-card">

## Vue d'ensemble en 30 secondes

```vue
<script setup lang="ts">
import { MrxGrid, type ColumnDef } from '@/components/MrxGrid'

const columns: ColumnDef[] = [
  { field: 'sku',   headerName: 'Référence', width: '120px', pinned: 'start' },
  { field: 'name',  headerName: 'Produit',   width: '260px', editable: true },
  { field: 'price', headerName: 'Prix',      width: '110px', sortable: true },
]
const rows = [
  { sku: 'LM-001', name: 'Perceuse 18V', price: 79.9 },
  { sku: 'LM-002', name: 'Tondeuse électrique', price: 199 },
]
</script>

<template>
  <MrxGrid :columns="columns" :rows="rows" virtual-scroll selectable />
</template>
```

    <tr>
      <td><strong>Groupement</strong></td>
